##  调用父类方法
### super()
1. 为了调用父类(超类)的一个方法，可以使用 `super()` 函数

```
class A:
    def spam(self):
        print('A.spam')

class B(A):
    def spam(self):
        print('B.spam')
        super().spam()  # Call parent spam()

b = B()
b.spam()
''' 
输出结果
B.spam  先调用 B 的 spam
A.spam  然后再调用 A 的 spam
'''
```
